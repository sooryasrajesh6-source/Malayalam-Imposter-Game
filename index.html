Find The Imposter

Malayalam Party Game

How to Play

1. Add minimum 3 players.

2. Each player secretly sees one Malayalam word.

3. Most players get the same word.

4. One player gets a different word. That player is the imposter.

5. Discuss and ask questions.

6. Vote the suspected imposter.

7. If the imposter is voted out, players win.
